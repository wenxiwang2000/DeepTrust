# OmniParser BBox Clicker

**OmniParser BBox Clicker** is a lightweight Python tool for fast, manual bounding-box (ROI) annotation.  
It allows users to click-and-drag to define regions of interest on images, provides immediate visual feedback, and exports bounding-box coordinates for downstream processing or model training workflows.

---

## Features

- Interactive click-and-drag bounding box selection  
- Real-time visual overlay  
- Exports coordinates (x1, y1, x2, y2)  
- Lightweight and fast (no heavy annotation frameworks)  
- Suitable for dataset preparation and vision-model preprocessing
